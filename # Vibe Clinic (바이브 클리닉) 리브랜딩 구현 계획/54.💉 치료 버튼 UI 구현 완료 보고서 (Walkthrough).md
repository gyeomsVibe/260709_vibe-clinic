## 2. 변경한 것

### dashboard-ui/src/App.jsx

**State 추가**
- `isCureAllRunning`: 치료 진행 중 UI 잠금
- `cureAllReport`: 치료 리포트 모달 데이터

**runCureAll() 함수**
- `POST /api/repair/cure-all` 호출 (strategy: 'auto')
- 결과를 `cureAllReport`에 저장 → 모달 자동 오픈
- 완료 후 자동 재진단 + 치료 원장 갱신

**💉 치료 버튼 (툴바)**
- 진단 실행 🩺 옆 배치
- 실패(ERROR+WARNING) ≥ 1건: 빨간 글로우 활성 상태
- 실패 0건: 반투명 비활성 (opacity 0.45)
- 치료 중: 빨간 스피너 + "치료 중…" 텍스트
- 진단 실행과 교차 비활성화

**치료 리포트 모달**
- 5분류 카운터 그리드: ✅완치 / ⚠️롤백 / 📋수동조치 / 🚫약화차단 / 🔑처방불가
- 완치 목록: diagId, "재진단 OK ✓" 배지, 처방 요약, 수정 파일
- 롤백 목록: diagId, 회귀 대상 목록
- 수동 조치 목록: diagId, 처방전 단계별 표시
- 처방불가 목록: diagId 표시
- 헤더에 "완치 인정 기준: VERIFIED_RESULT만 — 할루시네이션 치료 차단" 명시
