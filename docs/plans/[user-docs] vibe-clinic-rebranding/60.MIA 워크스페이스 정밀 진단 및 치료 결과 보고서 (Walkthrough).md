### 🔒 대시보드 보안 장치 정밀 진단 (G1 ~ G3)
[dashboard.js](file:///d:/D_Workspace_NB/-google-workspace/-antigravity-workspace/260709_vibe-clinic/backend/src/dashboard.js) 소스 코드를 대상으로 한 보안 맹점 검사 결과는 다음과 같습니다:

- **[G1] Origin null 403 차단**: `isAllowedDashboardOrigin` 미들웨어 검증을 통해 외부 웹이나 `file://` 등의 `null` 오리진에서 비정상적으로 POST 요청을 주입하는 행위를 원천 차단하고 있습니다.
- **[G2] 경로 탈출(Directory Traversal) 차단**: `isInsideDist` 검증을 거쳐 `path.resolve` 결과가 `dist-v2` 또는 `dist` 폴더 내부가 아닌 경우 무조건 403 Forbidden을 응답하게 설계되어 있습니다.
- **[G3] API 페이로드 1MiB 제한**: `MAX_BODY_BYTES = 1024 * 1024` 설정을 통해, `readBody` 수신 시 이 범위를 초과하는 악성 JSON 요청에 대해서 즉시 413 Payload Too Large 오류를 내며 커넥션을 차단합니다.
- **[E1 ~ E2] 오류 패턴 노출**: `.vibe-clinic/error-patterns` 하위의 파일만 안전하게 조회하도록 `path.basename`으로 파일명을 세니타이징하고 있어 안전합니다.
