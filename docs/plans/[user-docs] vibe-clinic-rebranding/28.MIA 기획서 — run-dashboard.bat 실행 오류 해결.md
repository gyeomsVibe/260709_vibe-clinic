## 1. Opportunity Brief

### 해결하려는 문제

#### 문제 ①: `EADDRINUSE: address already in use 127.0.0.1:7700`
- **원인**: 이전 턴에서 백그라운드로 실행해 둔 Node.js 서버 프로세스가 7700 포트를 점유하고 있어, 사용자가 `run-dashboard.bat`을 실행할 때 충돌이 발생했습니다.
- **해결**: 백그라운드 프로세스를 즉시 종료하여 포트 점유를 해제했습니다.

#### 문제 ②: `'????????????...'은(는) 내부 또는 외부 명령...` 오류
- **원인**: Windows CMD의 기본 한국어 코드페이지(CP949)가 UTF-8로 저장된 `run-dashboard.bat` 내의 유니코드 박스 문자(`────────────────`)를 제대로 해독하지 못하고 잘못된 명령어로 오인하는 현상입니다.
- **해결**: 유니코드 특수 선 문자를 표준 ASCII 하이픈(`----------------------------------`)으로 변경하여 인코딩 충돌을 원천 차단합니다.
