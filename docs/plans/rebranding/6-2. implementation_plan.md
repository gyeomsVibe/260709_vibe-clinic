# Vibe Clinic: 폴더 선택 대화상자 연동 및 에러 패턴 동적 갱신 보강 계획

대시보드 상에서 수동으로 텍스트 경로를 기입하던 방식을 윈도우 폴더 선택 대화상자(Folder Dialog)를 통한 원클릭 기동 방식으로 고도화하고, 프로젝트 전환 시 에러 패턴과 헬스 스탯이 실시간으로 교체 렌더링되지 않던 캐시 및 이벤트 꼬임 문제를 해결합니다.

---

## 1. 보강되는 변경 사항

### ① 백엔드 폴더 대화상자 구현 ([src/dashboard.js](file:///d:/D_Workspace_NB/-google-workspace/-antigravity-workspace/260709_vibe-clinic/src/dashboard.js))
- **신규 API 설계 (`POST /api/project/select`)**:
  - 프론트엔드에서 **[📁 폴더 선택]** 버튼을 누르면 이 API가 호출됩니다.
  - 백엔드는 윈도우 OS의 파워셸 COM 개체를 기동하여 사용자 화면에 네이티브 폴더 선택창을 팝업시킵니다.
    ```powershell
    Add-Type -AssemblyName System.Windows.Forms;
    $dialog = New-Object System.Windows.Forms.FolderBrowserDialog;
    $dialog.Description = 'Vibe Clinic 검증 대상 폴더를 선택하세요';
    if ($dialog.ShowDialog() -eq 'OK') { Write-Host $dialog.SelectedPath }
    ```
  - 파워셸 프로세스의 표준 출력(stdout)을 파싱하여 선택된 절대 경로를 프론트엔드로 응답합니다.

### ② 에러 패턴 리팩토링 및 캐시 우회 ([src/dashboard.html](file:///d:/D_Workspace_NB/-google-workspace/-antigravity-workspace/260709_vibe-clinic/src/dashboard.html))
- **폴더 선택 UI 버튼 결합**: 텍스트 경로 입력창 우측에 **[📁 폴더 선택]** 단추를 매핑하여 원클릭으로 팝업을 기동시킵니다.
- **에러 패턴 렌더러 캐시 파괴**:
  - 프로젝트 전환 시 `loadErrors()`가 완전히 동기적으로 다시 호출되도록 정비합니다.
  - 동일한 템플릿 파일명(예: `ERR_000_template.md`)을 읽을 때 브라우저가 이전 메모리를 재사용하지 못하도록, API 조회 시 타임스탬프(`?t=Date.now()`)를 바인딩하여 핫스왑 즉시 에러패턴 문서 내용이 실시간 갱신되도록 보장합니다.
