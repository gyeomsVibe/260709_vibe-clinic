# MIA 코어 로직 P1~P5 전체 완료 보고서 (Walkthrough)

> **문서 상태:** `VERIFIED_RESULT` (자동 테스트 39/39 + 260714 실서버 E2E 검증 완료)
> **기준 기획:** 46번 마스터플랜 — 진단·치료 코어 로직 MIA 모드 접목

---

## 1. 요약 — 무엇이 바뀌었나

Vibe Clinic의 진단→처방→치료 파이프라인이 MIA의 "증거 기반 진단 → 후보 비교 → 최소 수정 → 검증 → 학습" 사이클을 코드로 수행하도록 5단계(P1~P5)를 모두 접목했다. "대시보드 디자인보다 정밀 진단·완벽 치료 로직이 핵심"이라는 방향에 맞춰 코어 로직을 강화했다.

## 2. 단계별 구현 내역

### P1 — 정밀 진단 (Evidence Gate) · 커밋 854eda4
- **Flaky Gate**: 실패 진단을 자동 1회 재실행 → 2회 재현되면 `CONFIRMED(확진)`, 1회만이면 `SUSPECTED(간헐 의심, 재현 1/2)`. 부작용 있는 진단은 `retriable:false`로 옵트아웃.
- **triage.js**: 증상→원인 후보 규칙 엔진(ESM/CJS 불일치·모듈 미설치·타임아웃·네트워크/쿼터·모듈 결함·로직 실패). 실패 결과에 `causeHypotheses`(≤3, 가능성순) 부착. repairer의 문자열 감지 로직을 이 엔진으로 일원화.

### P2 — 처방 포트폴리오 + 판단 계약 · 커밋 8e3c8a3
- **진단 약화 차단 가드(BLOCKED_WEAKENING)**: 처방이 진단 파일(.clinic.*)을 수정하려면 원인이 "진단 파일 자체 결함"일 때만 허용, 그 외엔 자동 거부. "가짜 완치 금지"를 코드로 집행.
- **처방 전략 3종**: `auto`(로컬 룰 우선, 없으면 AI) / `local`(무AI) / `ai`(AI 강제). 처방마다 평가 메타(`filesTouched·bytes·reversible·touchesDiagnostics`) 부착.

### 수동 처방전 (근본 원인 해소) · 커밋 7a2f451
- "처방·치료가 안 이루어진다"의 근본: 파일 수정으로 못 고치는 병(잔액 부족·네트워크·모듈 미설치)에 치료 경로가 아예 없었음.
- 진단이 `prescription` 필드로 조치법 제공 가능 + triage 원인 기반 조치 가이드. 모든 막다른 길(로컬 룰 없음·AI files:[]·약화 차단·API 실패)에 폴백. `kind:'MANUAL'` — 재진단으로 완치 확인.

### P3 — 안전 치료 (Regression Guard) · 커밋 854eda4
- 치료 후 **전체 재진단** → 대상 미완치 또는 이전 OK 진단 파손(회귀) 시 **자동 롤백**(.bak 복원 + 신규 파일 삭제).
- 성숙도 라벨: `STATIC_CANDIDATE → APPLIED / VERIFIED_RESULT / ROLLED_BACK`.

### P4 — 치료 원장 (학습 루프) · 커밋 7d235d8
- 모든 치료 결과를 `.vibe-clinic/treatment-ledger.json`에 append(런타임 상태, gitignore).
- VERIFIED_RESULT 치료는 `error-patterns/CURE_<diagId>.md` 자동 생성 → 같은 진단의 다음 실패 때 AI 프롬프트에 자동 주입(같은 병 재치료 시 재질문 불필요).
- `GET /api/treatments` 원장 조회 API.

### P5 — UI 노출 · 커밋 7d235d8
- 카드: `확진 (재현 2/2)` / `간헐 의심` 배지.
- 드로어: 확진/의심 판정 + `원인 후보`(가능성·원인·근거) 목록.
- 타임라인: 세션 임시 기록 → 영속 치료 원장(성숙도 라벨·전략) 연동.
- 적용 결과: `완치 검증 완료(회귀 0)` / `자동 롤백됨` 구분 토스트.

## 3. 부수 발견·수정 (robustness)
- **빈 카드 실패 모드**: `useIntersectionObserver`가 초기값 false여서 Observer가 안 fire되면 진단 카드가 영구 공백이 됐음. 초기값 true + Observer 미지원 시 no-op으로 수정(진단 대시보드에서 공백 카드는 치명적).
- **API 키 유출 직전 차단**(이전 커밋 03df3d4): 추적되던 example config.json에 실키 저장 → 추적 해제 + `**/.vibe-clinic/config.json` 무시.

## 4. 검증 결과
