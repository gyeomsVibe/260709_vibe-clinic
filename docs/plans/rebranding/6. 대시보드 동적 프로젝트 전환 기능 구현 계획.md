# Vibe Clinic: 대시보드 동적 프로젝트 전환 기능 구현 계획

대시보드 기동 시 고정된 디렉토리만 검증하던 한계를 해소하고, 웹 대시보드 화면상에서 현재 Vibe Clinic 프로젝트 본체와 하위 예제 프로젝트들(또는 사용자 정의 절대 경로)을 동적으로 선택 및 전환할 수 있는 기능을 추가합니다.

---

## 1. 제안하는 변경 사항

### 백엔드 고도화 ([src/dashboard.js](file:///d:/D_Workspace_NB/-google-workspace/-antigravity-workspace/260709_vibe-clinic/src/dashboard.js))
- **동적 경로 변수 도입**: `startDashboard` 구동 시 기본 타겟 디렉토리를 고정 상수가 아닌 동적 변수 `currentProjectDir`로 전환하여 런타임에 유연하게 덮어쓸 수 있도록 합니다.
- **신규 API 설계**:
  - `GET /api/project/list`: 대시보드가 가동 중인 저장소 루트 경로(`.`) 및 하위 예제 디렉토리 목록(`examples/*`)을 자동 스캔하여, 사용자가 선택할 수 있는 프로젝트 목록 리스트를 반환합니다.
    ```json
    [
      { "name": "Vibe Clinic 본체", "path": "D:/.../260709_vibe-clinic" },
      { "name": "예제: calculator", "path": "D:/.../260709_vibe-clinic/examples/calculator" }
    ]
    ```
  - `POST /api/project/change`: 프론트엔드로부터 변경할 타겟 절대 경로를 전송받아 `currentProjectDir`을 안전하게 갱신합니다. (물리적 폴더 유효성 검사 포함)

---

### 프론트엔드 UI/UX 고도화 ([src/dashboard.html](file:///d:/D_Workspace_NB/-google-workspace/-antigravity-workspace/260709_vibe-clinic/src/dashboard.html))
- **프로젝트 셀렉터(Selector) 컴포넌트 추가**:
  - 대시보드 상단 툴바(Toolbar) 영역에 **"프로젝트 선택"** 드롭다운(Dropdown) 메뉴를 배치합니다.
  - 드롭다운 우측에 사용자가 임의의 다른 프로젝트 절대 경로를 직접 기입할 수 있는 인풋(Input) 상자 및 **[이동]** 버튼도 옵션으로 배치해 범용성을 높입니다.
- **상태 동기화 및 렌더링**:
  - 드롭다운 또는 인풋 경로 변경 시 `POST /api/project/change` API를 호출하고, 성공 시 대시보드를 즉시 새로고침(`init()` 및 `runDiagnostics()`)하여 전환된 프로젝트의 진단 카드를 실시간 렌더링합니다.
